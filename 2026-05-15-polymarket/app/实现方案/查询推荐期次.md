# 接口

## 查询推荐期次（已完成）
* 表oracle_issue
* 返回List,不需要分页
* 查询表中is_recommended=1并且status=1的数据,倒序
## 请求参数
* symbol 币对
### 响应字段
* ID id
* issue_code 期次编码
* symbol 币对
* baseCoin 基础币
* begin_time 开始时间
* end_time 结束时间
* buyBeginTime 购买开始时间
* buyEndTime 购买结束时间
* currentTime 当前时间
* klinePeriod K线周期
* open_price 基准价
* close_price 结束价
* start_advance_minutes 参与开始提前分钟数
* cutoff_advance_minutes 参与截止提前分钟数
* marketType 市场类型
* status 状态(1=进行中,2=待开始,3=待开奖,4=已开奖)
* statusName 状态名
* is_recommended是否推荐(1=是,2=否)
* total_effective_amount 有效参与金额
* positive_amount 正方(涨方)金额
* negative_amount 反方(跌方)金额
