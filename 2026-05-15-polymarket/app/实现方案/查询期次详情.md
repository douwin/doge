# 接口
本接口使用Get请求

## 查询期次详情（已完成）
* 表oracle_issue
### 请求参数
ID 期次ID
### 响应字段
* ID id
* issue_code 期次编码
* symbol 币对
* baseCoin 基础币
* begin_time 开始时间
* end_time 结束时间
* open_price 基准价
* close_price 结束价
* start_advance_minutes 参与开始提前分钟数
* cutoff_advance_minutes 参与截止提前分钟数
* marketType 市场类型
* status 状态(1=进行中,2=待开始,3=待开奖,4=已开奖)
* statusName 状态名
* fee_rate 手续费率
* total_effective_amount 有效参与金额
* positive_amount 正方(涨方)金额
* negative_amount 反方(跌方)金额
* limit_mode 限制模型(1=仅1次,2=不限制)
* allocType 净清算池类型
* allocTypeName 净清算池名称
* buyBeginTime 购买开始时间
* buyEndTime 购买结束时间
* buyInfo 购买信息(集合,如果未登录返回空集合)
*    choiceValue 涨/跌
*    amount 数量
* balance USDT余额(如果未登录返回空)
* isParticipate 是否参与(true=是,false=否,如果未登录返回false)
