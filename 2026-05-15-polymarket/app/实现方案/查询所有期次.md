# 接口

## 查询所有期次（已完成）
* 表oracle_issue
* 分页查询
* 该接口使用Get请求
### 请求参数
status 状态(1=进行中,2=待开始,3=待开奖,4=已开奖)
* symbol 币对
### 响应字段
* ID id
* issue_code 期次编码
* symbol 币对
* baseCoin 基础币
* begin_time 开始时间
* end_time 结束时间
* buyBeginTime 购买开始时间
* buyEndTime 购买结束时间
* currentTime 当前时间
* open_price 基准价
* close_price 结束价
* start_advance_minutes 参与开始提前分钟数
* cutoff_advance_minutes 参与截止提前分钟数
* marketType 市场类型
* status 状态(1=进行中,2=待开始,3=待开奖,4=已开奖)
* statusName 状态名
* draw_result 开奖结果(1=正方胜,2=反方胜,非已开奖时返回空)
* income_amount 收益金额(status=4并且登录时有效,否则返回空)
