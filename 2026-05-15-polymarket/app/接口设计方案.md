# 开发
1. 数据库:mysql
2. 开发语言:java

# 要求
1. 执行“连接数据库”加载skill,通过提供的表读取表结构
2. 一定要遵循现有代码的规范
3. 关键位置和方法需要中文注释
4. SQL需要有一个很好的可读性格式
5. 写代码时一直执行，无需我确认
6. 禁止修改表结构,如果必须修改表结构只给出sql语句
7. 返回的时间、Long 、BigDecimal 都必须转化成String类型,时间必须转化成时间戳返回
8. 非特殊说明全部使用post请求
9. 不能修改现有业务逻辑
11. 所有参数和响应字段都需要使用驼峰命名
12. 请求中除tinyint、Int、BigDecimal 外其他全部用string类型

# 接口
## 查询交易对
请求方式: Get
* 根据枚举OraclePairSymbolEnum返回对应的数据
* 返回模型SelectRes

## 查询推荐期次
请求方式: Get
* 表oracle_issue
* 返回List,不需要分页
* 查询表中is_recommended=1并且status=1的数据,倒序
## 请求参数
* symbol 币对
### 响应字段
* ID id
* issue_code 期次编码
* symbol 币对
* baseCoin 基础币
* begin_time 开始时间
* end_time 结束时间
* open_price 基准价
* close_price 结束价(当前价)
* start_advance_minutes 参与开始提前分钟数
* cutoff_advance_minutes 参与截止提前分钟数
* status 状态(1=进行中,2=待开始,3=待开奖,4=已开奖)
* statusName 状态名
* is_recommended是否推荐(1=是,2=否)
* total_effective_amount 有效参与金额
* positive_amount 正方(涨方)金额
* negative_amount 反方(跌方)金额
* marketType 市场类型
* buyBeginTime 购买开始时间
* buyEndTime 购买结束时间

## 查询期次详情
请求方式: Get
* 表oracle_issue
### 请求参数
ID 期次ID
### 响应字段
* ID id
* issue_code 期次编码

* begin_time 开始时间
