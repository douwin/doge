# 1. 查询已启用的club
* 请求方式
GET
* 参数
无
* 响应
clubConfigId Club配置ID
name 名称
description 描述
benefitList 权益集合(List)
- name 名称
- unit 单位
- num 数量
isBuy 是否购买

memberInfo 会员信息
- memberId 会员ID
- memberNo 会员卡号
- valid_start 有效期(开始时间)
- valid_end 有效期(结束时间)
