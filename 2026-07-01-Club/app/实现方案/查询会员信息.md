# 查询会员信息

## 接口信息
| 项 | 内容 |
|---|---|
| 请求地址 | `/api/v1/club/member/info` |
| 请求方式 | `GET` |
| Token | 必填 |

## 业务说明
1. 返回当前登录账号最新一条 Club 会员记录。
2. 当前实现按 `club_member.update_time desc, id desc` 取最新会员，不额外传 `clubConfigId`。

## 请求参数
无

## 请求示例
```http
GET /api/v1/club/member/info
Language: zh_CN
Token: eyJhbGciOiJIUzI1NiJ9.demo
```

## 响应数据
`data` 为对象

| 字段 | 类型 | 说明 |
|---|---|---|
| `clubConfigId` | String | Club配置ID |
| `clubConfigName` | String | Club名称 |
| `memberId` | String | 会员ID |
| `memberNo` | String | 会员号 |
| `validStart` | String | 生效开始时间戳(毫秒) |
| `validEnd` | String | 生效结束时间戳(毫秒) |
| `status` | Integer | 会员状态：`1=正常`，`2=已超期`，`3=已冻结`，`4=已停用`，`5=已失效` |
| `statusName` | String | 会员状态名称 |
| `admissionFee` | String | 入会费 |
| `annualFee` | String | 年费 |
| `createTime` | String | 开通时间戳(毫秒) |
| `availablePoints` | String | 可用积分，取 `club_member.points` |
| `availableTickets` | String | 可用票据，取 `club_member.ticket_num` |
| `benefitList` | Array | 会员权益快照集合 |

`benefitList` 元素结构：

| 字段 | 类型 | 说明 |
|---|---|---|
| `id` | String | 会员权益ID |
| `benefitType` | Integer | 权益类型：`1=ELON代币`，`2=高级权益票证`，`3=俱乐部积分`，`4=预测平台分红`，`5=矿池权益` |
| `name` | String | 权益名称 |
| `unit` | String | 单位 |
| `num` | String | 数量，取 `club_member_benefit.config_num` |
| `status` | Integer | 权益状态 |
| `statusName` | String | 权益状态名称 |

## 校验说明
1. 本接口要求登录。
2. 当前账号不存在任何会员记录时，返回 `CLUB_MEMBER_NOT_EXISTS`。
3. 会员关联的 Club 配置不存在时，返回 `CLUB_NOT_EXISTS`。

## 复杂业务说明
1. 会员状态不是直接读表返回，而是动态计算：
   - 若原始状态为已冻结、已停用、已失效，直接返回该状态。
   - 否则在 `validEnd` 之前返回正常。
   - 超过 `validEnd` 后，若仍在宽限期内返回已超期。
   - 超过宽限期后，若 `post_grace_action=新开卡` 返回已失效，否则仍返回已超期。
2. `benefitList` 只读取 `club_member_benefit` 快照表；如果快照未生成，则直接返回空数组，不回退到 `club_config_benefit`。
3. `benefitList.statusName` 使用“已生效/待生效”枚举解释。

## 失败码
| 错误码 | 说明 |
|---|---|
| `10049` | Club不存在 |
| `10054` | Club会员不存在 |
